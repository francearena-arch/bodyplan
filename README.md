# BodyPlan PWA

## Installation über GitHub Pages
1. Neues GitHub Repository erstellen, z. B. `bodyplan`.
2. Alle Dateien aus diesem Ordner hochladen.
3. In GitHub unter Settings → Pages → Branch `main` und `/root` auswählen.
4. Die generierte URL in Safari auf dem iPhone öffnen.
5. Teilen-Symbol → Zum Home-Bildschirm.

## Enthalten
- iPhone-fähige Web-App
- Offline-Funktion über Service Worker
- App-Icon
- Lokale Speicherung von Gewicht und Körperfett
- Checkboxen für absolvierte Workouts
- Wochen-Reset Button
